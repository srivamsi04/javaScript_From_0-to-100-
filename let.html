JavaScript let Keyword (ES6 / 2015)

1. Introduced in ES6 (2015)


let x = 10;
console.log(x); // 10

2. Block Scope


{
    let a = 5;
    console.log(a); // 5
}
console.log(a); // Error: a is not defined

3. Must be declared before use


console.log(b); // Error: Cannot access 'b' before initialization
let b = 20;

4. Cannot be redeclared in the same scope


let c = 10;
let c = 20; // Error: Identifier 'c' has already been declared

But you can declare it in a different block:


let d = 1;
{
    let d = 2; // Different block scope
    console.log(d); // 2
}
console.log(d); // 1

5. Key Differences Between var and let

Feature var let
Scope Function Block
Hoisting Yes (initialized as undefined) Yes, but cannot use before declaration (temporal dead zone)
Redeclare in same scope Allowed Not allowed
