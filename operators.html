JavaScript Operators

Operators are used for mathematical, logical, and assignment computations.

1. Assignment Operator (=)

The assignment operator assigns a value to a variable:


// Examples
let x = 10; // Assign 10 to x
let x2 = 5; // Assign 5 to x2
let y = 2;  // Assign 2 to y
let z = x2 + y; // Assign sum of x2 + y to z
console.log(z); // 7

2. Addition Operator (+)

Adds numbers or concatenates strings:


// Numbers
let a = 5;
let b = 2;
let sum = a + b; 
console.log(sum); // 7

// Strings
let text1 = "John";
let text2 = "Doe";
let fullName = text1 + " " + text2;
console.log(fullName); // "John Doe"

Using += for Addition/Concatenation


let text = "What a very ";
text += "nice day";
console.log(text); // "What a very nice day"

Adding Numbers and Strings


let x1 = 5 + 5;      // 10
let y1 = "5" + 5;    // "55"
let z1 = "Hello" + 5; // "Hello5"
console.log(x1, y1, z1);

3. Multiplication Operator (*)


let x = 5;
let y = 2;
let result = x * y;
console.log(result); // 10

4. Arithmetic Operators


let a = 3;
let total = (100 + 50) * a; // 450
console.log(total);

// Common arithmetic operators
// + Addition
// - Subtraction
// * Multiplication
// ** Exponentiation
// / Division
// % Modulus
// ++ Increment
// -- Decrement

5. Assignment Operators


let x = 10;
x += 5; // x = x + 5
x -= 3; // x = x - 3
x *= 2; // x = x * 2
x /= 4; // x = x / 4
x %= 3; // x = x % 3
x **= 2; // x = x ** 2
console.log(x);

6. Comparison Operators

Compare values and return true or false:


let x = 5;
console.log(x == 5);   // true
console.log(x === 5);  // true
console.log(x != 5);   // false
console.log(x !== 5);  // false
console.log(x > 5);    // false
console.log(x < 5);    // false
console.log(x >= 5);   // true
console.log(x <= 5);   // true

// String comparison
let text1 = "A";
let text2 = "B";
console.log(text1 < text2); // true (alphabetical comparison)

7. Logical Operators


let a = true;
let b = false;

console.log(a && b); // false (logical AND)
console.log(a || b); // true  (logical OR)
console.log(!a);     // false (logical NOT)

8. Types of Operators in JavaScript

  • Arithmetic Operators (+, -, *, /, %, **, ++, --)
  • Assignment Operators (=, +=, -=, *=, /=, %=, **=)
  • Comparison Operators (==, ===, !=, !==, >, <, >=, <=)
  • Logical Operators (&&, ||, !)
  • Others (Bitwise, Conditional, typeof, instanceof, etc.)
